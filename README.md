The site includes specific mobile optimizations:

- Reduced header parallax height (20vh on mobile vs 50vh on desktop)
- Flush parallax sections with no gaps between pink sections and images
- Stacked layout for white sections on mobile
- Larger, more readable ticker tape text
- Touch-friendly buttons and interactive elements
- Performance optimizations for mobile devices
